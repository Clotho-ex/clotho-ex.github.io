HTML Table CheatSheet

Table Tags

Table Tag

Stands For

Description
<table> Table Element Defines a table and wraps all table elements.
<th> Table Header Element Defines a header cell in a table.
<tr> Table Row Element Defines a row in a table.
<td> Table Data Element Defines a cell in a table.
<thead> Table Head Element Groups the header content in a table.
<tbody> Table Body Element Groups the body content in a table.
<tfoot> Table Footer Element Groups the footer content in a table.
<caption> Caption Element Used to insert captions.
<col> Column Element Defines a column within a table.
<colgroup> Column Element Defines a group of columns within a table.
